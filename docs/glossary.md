# Glosario del curso

## A

**Aprendizaje automático (Machine Learning)**
Rama de la IA que permite a los sistemas aprender patrones a partir de datos sin ser programados explícitamente.

**Aprendizaje supervisado (Supervised Learning)**
Paradigma de ML donde el modelo aprende a partir de datos etiquetados (features + target).

**Aprendizaje no supervisado (Unsupervised Learning)**
Paradigma de ML donde el modelo encuentra patrones en datos no etiquetados.

## B

**Bias (sesgo)**
1. En estadística: diferencia sistemática entre el valor estimado y el valor real.
2. En ética: prejuicio sistemático en algoritmos que produce resultados injustos.

## C

**Clasificación**
Tarea de ML supervisado donde la variable target es categórica.

**Clustering**
Tarea de ML no supervisado que agrupa observaciones similares.

## D

**Dataset**
Conjunto de datos estructurados utilizado para entrenar y evaluar modelos.

**Deep Learning**
Subcampo del ML basado en redes neuronales con múltiples capas.

## E

**EDA (Exploratory Data Analysis)**
Proceso de analizar datasets para resumir sus características principales usando visualizaciones y estadísticas.

**Feature**
Variable de entrada utilizada por un modelo para hacer predicciones.

## G

**Generalización**
Capacidad de un modelo de ML para funcionar correctamente sobre datos no vistos durante el entrenamiento.

## K

**K-Means**
Algoritmo de clustering que particiona n observaciones en k clusters.

## L

**Label (etiqueta)**
Valor de salida correcto en datos de entrenamiento supervisado.

**Linear Regression**
Modelo de regresión que asume una relación lineal entre features y target.

## M

**Overfitting**
Problema donde el modelo aprende el ruido de los datos de entrenamiento y no generaliza a datos nuevos.

**Underfitting**
Problema donde el modelo es demasiado simple para capturar la estructura de los datos.

## P

**PCA (Principal Component Analysis)**
Técnica de reducción de dimensionalidad que transforma variables correlacionadas en componentes no correlacionados.

## R

**Random Forest**
Algoritmo de ensemble learning que construye múltiples árboles de decisión y combina sus predicciones.

**Regresión**
Tarea de ML supervisado donde la variable target es continua.

**RMSE (Root Mean Square Error)**
Métrica de error que mide la diferencia promedio entre predicciones y valores reales.

## S

**SaaS (Software as a Service)**
Modelo de distribución de software donde las aplicaciones son alojadas por un proveedor y puestas a disposición de los clientes a través de internet.

**Train/Test Split**
Técnica que divide un dataset en dos partes: una para entrenar el modelo y otra para evaluarlo.
