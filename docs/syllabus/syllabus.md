# Syllabus

## Descripción del Curso

Curso introductorio de Machine Learning diseñado para estudiantes de Biotecnología, Bioinformática, Ciencias Biológicas, Análisis de Datos e Ingeniería. El curso combina fundamentos teóricos con implementación práctica en Python, utilizando ejemplos del mundo real en biotecnología y SaaS.

## Prerrequisitos

- Matemáticas de bachillerato
- Álgebra básica
- Sin experiencia previa en programación
- Sin experiencia previa en Machine Learning

## Objetivos de Aprendizaje

Al finalizar el curso, los estudiantes podrán:

1. Comprender los fundamentos de la Inteligencia Artificial
2. Escribir código Python para análisis de datos
3. Explorar y limpiar datasets usando pandas y numpy
4. Aplicar conceptos estadísticos a datos reales
5. Entrenar modelos de aprendizaje supervisado
6. Entrenar modelos de aprendizaje no supervisado
7. Evaluar modelos de Machine Learning
8. Interpretar outputs de modelos
9. Comunicar hallazgos mediante data storytelling
10. Resolver problemas de Biotecnología y SaaS usando Machine Learning
