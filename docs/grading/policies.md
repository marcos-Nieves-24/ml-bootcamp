# Grading Policies

## Assessment Components

| Component | Weight | Description |
|-----------|--------|-------------|
| Quizzes (5) | 15% | Weekly online quizzes after each module section |
| Labs (5) | 20% | In-class practical exercises |
| Assignments (3) | 20% | Take-home programming assignments |
| Midterm Exam | 15% | Written exam covering Modules 1-3 |
| Final Exam | 15% | Written exam covering Modules 4-5 |
| Final Project | 15% | End-to-end ML pipeline project |

## Grading Scale

| Grade | Percentage |
|-------|------------|
| A | 90-100% |
| B | 80-89% |
| C | 70-79% |
| D | 60-69% |
| F | <60% |

## Late Submission Policy

- Up to 24 hours late: -10%
- 24-48 hours late: -20%
- 48-72 hours late: -30%
- More than 72 hours: not accepted without prior arrangement

## Academic Integrity

- All submitted work must be the student's own
- Collaboration is encouraged for labs but assignments must be individual
- AI assistants may be used as learning tools but not to generate submissions
- Violations will result in a score of 0 for the assignment and referral to academic affairs

## Re-evaluation Policy

- Requests for re-evaluation must be submitted within 1 week of grade release
- Include written justification
- The entire assignment/exam may be re-evaluated, not just the contested section
