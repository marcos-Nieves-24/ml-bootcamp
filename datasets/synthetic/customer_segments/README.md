# customer_segments

## Descripción
Dataset sintético de segmentación de clientes para ejercicios de clustering. Contiene características demográficas y de gasto.

## Fuente
Generado sintéticamente con fines educativos.

## Variables

- `age` (int64)
- `annual_spend` (int64)
- `cluster` (int64)
- `membership_years` (float64)
- `num_purchases` (int64)
