# biotech_quality

## Descripción
Dataset sintético de control de calidad en manufactura biotech. Parámetros de proceso y resultados de calidad para producción por lotes.

## Fuente
Generado sintéticamente con fines educativos.

## Variables

- `batch_id` (int64)
- `temperature` (float64)
- `ph_level` (float64)
- `pressure` (float64)
- `stirring_speed` (float64)
- `culture_time_hours` (float64)
- `oxygen_saturation` (float64)
- `nutrient_concentration` (float64)
- `quality_score` (float64)
- `pass_quality` (int64)
