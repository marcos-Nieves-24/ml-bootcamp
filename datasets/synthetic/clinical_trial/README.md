# clinical_trial

## Description
Synthetic clinical trial dataset for data analysis exercises. Patient demographics, treatment assignment, and outcomes.

## Source
Generated synthetically for educational purposes.

## Variables

- `patient_id` (int64)
- `age` (int64)
- `sex` (str)
- `treatment_group` (str)
- `baseline_bp` (float64)
- `baseline_bmi` (float64)
- `baseline_glucose` (float64)
- `adherence_rate` (float64)
- `outcome_improvement` (float64)
- `responder` (int64)
