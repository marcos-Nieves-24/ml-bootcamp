# Slides: Data Storytelling

## Diapositiva 1: Title
- Data Storytelling
- Module 3, Lesson 10

## Diapositiva 2: Learning Objectives
- Structure a data story
- Apply visualization best practices
- Design effective dashboards
- Communicate to non-technical audiences

## Diapositiva 3: Why Data Storytelling?
- Best analysis is worthless if not understood
- Drives decision-making
- Bridges gap between data and action
- Essential skill for data scientists

## Diapositiva 4: The Data Story Arc
- Hook: Why should I care?
- Context: What data?
- Conflict: What problem?
- Analysis: What did we find?
- Resolution: What now?

## Diapositiva 5: Tufte's Principles
- Data-ink ratio: maximize data ink
- Lie factor: visual effect = data effect
- Avoid chartjunk (3D, excessive colors)

## Diapositiva 6: Choosing the Right Chart
- Comparison: bar chart
- Trend: line chart
- Distribution: histogram
- Relationship: scatter plot
- Composition: stacked bar

## Diapositiva 7: Color Best Practices
- Use colorblind-friendly palettes
- Sequential: ordered data
- Diverging: data with midpoint
- Qualitative: categories (≤8)
- Avoid red-green

## Diapositiva 8: Dashboard Design
- Hierarchy: most important top-left
- Consistency: colors, fonts, layout
- Context: benchmarks, targets
- Clean, uncluttered

## Diapositiva 9: Know Your Audience
- Executives: high-level, recommendations
- Scientists: methodology, details
- Public: simple, engaging
- Tailor depth and terminology

## Diapositiva 10: Key Takeaways
- One chart = one message
- Maximize data-ink ratio
- Tell a story with arc
- Design for your audience
- Aesthetics support function
