# Subunit 02: Unsupervised Learning

**Lessons:** L6 (K-Means Clustering), L7 (PCA)

## Learning objectives

After completing this subunit, students will be able to:

- Explain the difference between supervised and unsupervised learning
- Implement K-Means clustering with scikit-learn
- Select the optimal number of clusters using elbow method and silhouette score
- Apply PCA for dimensionality reduction and visualization
- Interpret explained variance ratio and loadings

## Prerequisites

Module 3 statistics (variance, covariance, correlation), Subunit 01 (ML fundamentals)

## Key concepts

- Unsupervised learning
- K-Means algorithm, centroid initialization
- Inertia, elbow method, silhouette coefficient
- Principal Component Analysis
- Eigenvalues, eigenvectors, explained variance
- Dimensionality reduction for visualization

## Software

scikit-learn, numpy, pandas, matplotlib, seaborn
